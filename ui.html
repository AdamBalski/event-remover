AGH lectures remover (https://plan.agh.edu.pl/ links)

Paste your ical link from plan.agh.edu.pl, and this will give you a new link. That link points to the same calendar but with all lectures and placeholder events removed.

Default filter: NOT "Wykład" AND NOT "Blokada" (an event is removed if any field contains those phrases).

Optional query supports quoted phrases, parentheses, and operators NOT, AND, OR. Parsing order: NOT > AND > OR (use parentheses to override). Example: exclude labs or exams: NOT ("Lab" OR "Exam").
